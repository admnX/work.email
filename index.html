Programming Languages

Languages I learned about In school

Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its use of significant indentation.
My experience with Python has been that it is both user-friendly and easy to learn, with a clear and straightforward syntax.

JavaScript

JavaScript is a versatile, high-level programming language primarily used for creating interactive and dynamic content on websites.
My experience with JavaScript has been challenging, and I continue to struggle with it. However, I am determined to improve and look forward to mastering the implementation of animations using JavaScript in the future.

HTML

HTML, or HyperText Markup Language, is the standard language used to create web pages. It defines the structure of a webpage using a series of elements and tags. These elements tell the browser how to display content such as headings, paragraphs, links, images, and more.
My experience with HTML has been exceptionally positive. I found it to be the easiest language to learn, and I thoroughly enjoyed the process of understanding and utilizing its features.

CSS

CSS, or Cascading Style Sheets, is a language used to style and layout web pages. It allows you to control the appearance of HTML elements, such as fonts, colors, spacing, and positioning.
My experience with CSS has been highly rewarding. It provided me with creative freedom, allowing me to style websites according to my preferences and make them visually appealing.
